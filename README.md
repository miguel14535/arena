# Dojo Control - PDV para Academia de Jiu-Jitsu

Sistema PDV local para academia de Jiu-Jitsu, funcionando com React + TypeScript + Vite + Tailwind e localStorage.

## Como rodar

```bash
npm install
npm run dev
```

Depois acesse o endereço exibido no terminal, normalmente:

```txt
http://localhost:5173/
```

## Funcionalidades

- Caixa com código de barras
- Carrinho e finalização de venda
- Pagamento por Pix, dinheiro ou cartão
- Cadastro de produtos da academia
- Cadastro de alunos/sócios com faixa e plano
- Histórico de vendas
- Relatórios de faturamento e estoque
- Dados salvos no navegador usando localStorage
